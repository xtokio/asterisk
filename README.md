# asterisk

Basic Asterisk wrapper of the AMI and some console commands to interact with an Asterisk server

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  asterisk:
    github: xtokio/asterisk
```

## Usage

```crystal
require "asterisk"
```
